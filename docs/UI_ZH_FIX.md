# v1.1.1 中文界面与布局修复

本次修复针对 Windows 真机截图暴露出的实际问题：

- 全部主界面文案改为中文
- 左侧栏固定为 264px，避免品牌和来源名称被截断
- 顶部标题与操作按钮拆分为两行，避免按钮挤压标题
- 对话列表固定 440px 宽，列标题和更新时间不再重叠
- 工作区取消可随意压缩的 PanedWindow，改为稳定的双栏 Grid 布局
- 预览区空状态、加载状态、错误状态全部中文化
- TRAE 密钥助手完整中文化
- 导出对话、批量导出、状态栏和错误提示全部中文化
- 修复腾讯 Marvis 在旧数据库结构中缺少 `model_id` 字段导致预览崩溃的问题

## 真机验收重点

1. 100%、125%、150% DPI 下左侧栏不得截断
2. 1440×900 和 1280×720 下顶部按钮不得覆盖标题
3. 腾讯 Marvis 对话预览不得再出现 `no such column: model_id`
4. 五个来源的切换、搜索、预览、单条导出和批量导出需正常
5. 所有用户可见文案应为中文
